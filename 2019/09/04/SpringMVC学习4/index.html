SpringMVC会将上传的文件绑定到MultipartFile对象中，可以通过transferTo()方法将文件存储到硬盘中MultipartFile对象中的常用方法如下：

  • void transferTo(File file)：将上传文件保存到指定的目录中
  • String getOriginalFilename()：获取上传文件的原名
  • String getContentType()：获取文件的MIME类型，比如image/jpeg等
  • boolean isEmpty()：是否有上传文件
